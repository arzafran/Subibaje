# Subibaje
SUBE-like app for public institutions in Mar del Plata.

## Steps to clone
> git clone http://github.com/arzafran/Subibaje.git

## Get to frontend folder
> cd subibaja

## Install dependencies
> npm install


> bower install

## Build the app
> grunt build

compiles less, and uglyfies js

## Watch files for changes
> grunt watch
